# Telegram MVP-бот для поиска тиммейта

Простой бот на Python + Telegram Bot API (через `aiogram`) для сценария:

1. Пользователь создаёт заявку (`/new_request`):
   - выбирает игру;
   - указывает время игры.
2. Бот рассылает заявку остальным зарегистрированным пользователям.
3. Получатели отвечают кнопками:
   - `Могу играть`
   - `Не могу`
4. Создатель видит статусы ответов (`/my_requests`).

## Что реализовано

- Пошаговый сценарий (FSM) создания заявки.
- Хранение пользователей, заявок и ответов в SQLite.
- Inline-кнопки для быстрых ответов.
- Отправка обновлений создателю заявки при каждом новом ответе.

## Структура

- `bot.py` — обработчики Telegram-команд, FSM, рассылка, кнопки.
- `storage.py` — слой хранения (SQLite).
- `.env.example` — пример переменных окружения.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# впишите BOT_TOKEN
python bot.py
```

## Команды бота

- `/start` — регистрация и справка.
- `/new_request` — создать новую заявку.
- `/my_requests` — список последних заявок и просмотр ответов.

## Ограничения MVP

- Используется polling (не webhook).
- Время игры вводится свободным текстом.
- Пользователь получает рассылки только после того, как хотя бы раз нажал `/start`.
